# Alfred V2 - SF18 NNUE
This repo is the 2nd version of Alfred, my chess AI. 
Contrary to the [1st version](https://github.com/Youn-T/NNUE-SF13) which was using an older version of stockfish (SF13), this one is using the strong an modern architecture of Stockfish 18.

## What is Stockfish ?
[Stockfish](https://github.com/official-stockfish/Stockfish) is a strong and popular chess engine allowing users to load custom nnue weights.
